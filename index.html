profillic by galosh

A suite of programs for estimating parameters of Profile HMMs using Baum-Welch, Conditional Baum-Welch, and both Unconditional and Conditional Quadratic Ascent.

See the README file.

Dependencies

  • BOOST
  • Seqan version 1.2
  • HMMoC-BFloat-Algebra
  • prolific
